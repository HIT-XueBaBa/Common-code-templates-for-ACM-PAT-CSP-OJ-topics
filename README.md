# ACM、PAT、CSP、OJ题目常用代码模板
**注：本仓库中涉及到一些向markdown文本中插入的latex数学公式，建议使用google浏览器并安装`MathJax Plugin for Github`插件进行浏览，否则无法执行latex渲染**
所附的代码模板是我在刷题过程中总结的比较常用的代码，大部分代码是我自己编写的**基于C++11语法的C++接口**，所以比较符合我自己的代码风格，解释和注释也不是很多。建立这个项目主要是做一下总结，以便自己以后使用，以后也会经常更新和修正。当然了，如果你觉得有些代码对你自己也有用处，也可以直接拿去使用，不过如果你不太习惯我的代码风格，使用起来可能会有些麻烦。如果有问题或者感觉我的代码中有bug，可以在`Issues`中留言哦ღ( ´･ᴗ･\` )比心
